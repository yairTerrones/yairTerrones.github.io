JSON JavaScript Object Notation.

JSON es una notación de objetos basada en JavaScript, utiliza una sintaxis que nos permite crear objetos de manera rapida y simple, estos objetos pueden ser utilizados de la manera que queramos y la notación se utiliza muy comúnmente.

Para escribir JSON debemos tener en cuenta lo siguiente: La creación de los objetos JSON implica escribir datos, para ello:

  • Los datos estan separados por comas.
  • los datos se escriban en pares, siendo primero el nombre o atributo del mismo y luego el valor del dato.
  • Los objetos JSON estan rodeados por llaves “{}”
  • Llaves cuadradas [] guardan arreglos, incluyendo otros objetos JSON

Ejemplo de un objeto JSON que guarda un usuario y password:

var objetoJSON = {"usuario":"user","password":"123456"};
