DOM (Document Object Model).

DOM permite a los programadores web acceder y manipular las páginas XHTML como si fueran documentos XML. De hecho, DOM se diseñó originalmente para manipular de forma sencilla los documentos XML.

A pesar de sus orígenes, DOM se ha convertido en una utilidad disponible para la mayoría de lenguajes de programación (Java, PHP, JavaScript) y cuyas únicas diferencias se encuentran en la forma de implementarlo.

Representa al documento HTML en forma de estructura de árbol, el documento se puede manipular facilmente a travéz de un arbol, es decir, agregar, eliminar, modificar elementos etc.

Podemos acceder de diferentes maneras:

document.getElementsByTagName('etiqueta')
Obtener todos los elementos en el documento con el nombre de la etiqueta se especifica.
document.getElementsByClassName('etiqueta')
Obtener todos los elementos con el nombre de la clase especificada.
document.getElementsById('etiqueta')
Obtener el elemento con el ID especificado.
