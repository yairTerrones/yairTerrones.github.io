Arreglos en JavaScript.

En los arreglos podemos realizar diversas cosas.

  • Creacion.
  • Acceso.
  • Propiedades.
  • Iteracion.
  • Caracteristicas especiales (Heterogeneos o Jagged(dentado))
  • Metodos

Creacion.

var arreglo = new Array();
var arreglo = [];
En estas formas anteriores creamos dos arreglo vacios.

Ejemplo de arreglos con datos:
var array = new Array('Hola','adios','cadena','vacios')

En los arreglos heterogeneos podemos tener mas de un tipo almacenado en el arreglo por ejemplo:
var array = new Array('Hola',true,5,'vacios')

Para acceder a un elemnto del arreglo usamos la notacion arreglo[indice], donde indice es la posicion del arreglo ala cual queremos acceder, esto nos cituara en la posicion que deseamos en el arreglo siempre y cuando exista en el arreglo.
Tomando el ejemplo anterior , var array = new Array('Hola',true,5,'vacios') si queremos acceder ala posicion 3, por ejemplo, esto se denotaria asi:

var aux = array[2]; , debemos recorar que los arreglos inician en cero (0), el valor de elemnto 3 se le asignara a la variable auxiliar aux.

Podemos tambien crear arreglos de arreglos y se crean de la siguiente manera:
var matriz = [[1,2,3],[4,5,6]]; ---> de esta manera tenemos un arreglo de arreglos.

Métodos.

En JavaScript podemos hacer uso de diferentes métodos como los siguientes:

Concat().
a.concat(b), este metodo nos devuelve otro arreglo.
Join()
Este metodo nos ayuda a unir todos los elementos de un arreglo en una cadena de caracteres.
push()
Este metodo agrega un elemento al final del arreglo.
pop()
Este metodo elimina el ultimo elemento del arreglo.
reverse()
Este metodo regresa un arreglo con los elementos invertidos (orden invertido)
sort()
Este metodo ordena los elementos de una manera ascendente tratandolos como cadenas.
shift()
Elimina el elemento que se encuentra al inicio.
unshift()
Inserta un elemento al inicio del arreglo.
splice()
Agrega o elimina.
slice()
Esxtrae un sub arreglo.
