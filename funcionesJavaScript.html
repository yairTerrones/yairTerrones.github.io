Funciones en JavaScript

Una función JavaScript es un bloque de código diseñado para realizar una tarea en particular. Una función de JavaScript se ejecuta cuando 'algo' lo invoca (llama).

Sintaxis.

Una función de JavaScript se define con la palabra clave función, seguido por un nombre, seguido de paréntesis (). Los nombres de funciones pueden contener letras, dígitos, guiones y símbolos de dólar (mismas reglas que las variables). Los paréntesis pueden incluir nombres de parámetros separados por comas: (Parámetro1, parámetro2, ...) El código que se ejecuta, por la función, se coloca entre llaves: { }

Ejemplo:

function nombre(parametro1, parametro2, parametro3) {
codigo que vamos a ejecutar....
}

Los parámetros de la función son los nombres que figuran en la definición de función, los argumentos de la función son los valores reales recibidos por la función cuando se invoca. Dentro de la función, los argumentos (los parámetros) se comportan como variables locales.
