Enlaces en HTML.

Los enlaces en HTML se crean mediante la etiqueta (su nombre viene del inglés "anchor", literalmente traducido como "ancla"). A continuación se muestra la definición simplificada de

Ejemplo de sintaxis:

Etiqueta < a > (anchor = ancla) Atributos: href = "enlace": Permite nombrar al enlace para que se pueda acceder desde otros enlaces.

target = _blank o _self: es la manera en como se va adesplegar la informacion en la pantalla, _blank se refiere a que la pagina se va a desplegar en la misma pagina, es decir, que no se abrirá un pantalla nueva, en caso de no especificarse nada, abrirá una nueva ventana para visualizar los datos.

Existen diferentes tipos de enlaces:

  1. Enlaces Relativos: los cuales se direccionan con una pagina que se encuentra en el mismo lugar que la pagina web.
  2. Absolutos: En una pestaña diferente.
  3. Contenido para descargar: Esta opcion no abre pestañas, es mas bien para archivos, cuando lo seleccionamos instantaneamente nos da el archivo.
