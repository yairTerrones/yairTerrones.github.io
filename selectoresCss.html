Selectores en CSS.

En CSS, los selectores son patrones utilizados para seleccionar los elementos que desea diseñar y existen diferentes tipos:

  1. Selector de tipo.
  2. Selector descendente.
  3. Selector hijo.
  4. Selector por ID.
  5. Selector por clase.
  6. Selector universal.

Selector de tipo: Selecciona un tipo de elemento HTMLy aplica el estilo a todos los elemntos.
Ejemplo:
p { text-aling: justify; }
Solo seleccionamos el tipo, en este caso un parrafo, y solo a elementos de ese tipo le aplicaremos el estilo.

Selector descendente: Aplica el estilo a elementos que estan contenidos dentro de otros elementos.
Ejemplo:

  1. p, b{ color: red; } En este caso solo se le aplica el estilo a bolt por estar dentro de un Parrafo
  2. ul li b {color: green; } En este caso se le aplica a los elemntos que contiene la lista desordenada.

Selector hijo: Similar al descendente pero mas restrictivo, aplica el estilo solamente a los elemntos que sean hijos directos de otro elemento.
Ejemplo:
p > b {color: blue; }

Selector por ID:Un id es un atributo de un elemento de HTML y se utiliza para aplicar u estilo de manera particular.
Ejemplo:
< p > id= "introduccion"> Este es el parrafo de la introduccion < /p >

Selector por clase: Utiliza el simbolo .
.clase{
instrucciones
}
Cuando usamos este selector todo lo que se ecuentre dentro de la clase sera editado y se le aplicara el estilo que esta definido en el archivo .css

< p > clase= "especial"> Este es el parrafo especial < /p >

Selector universal: Utiliza el simbolo *
* {
instrucciones
}
Cuando usamos este selector, el contenido que se encuentre dentro de el aplicara para todos lo elementos de HTML

< p > clase= "especial"> Este es el parrafo especial < /p >
