HTML (Hypertext Markup languaje) lenguaje de marcado de Hipertexto.

HTML es un lenguaje que se utiliza fundamentalmente en el desarrollo de páginas web. HTML es la sigla de HiperText Markup Language (Lenguaje de Marcación de Hipertexto) es un lenguaje es se utiliza comúnmente para establecer la estructura y contenido de un sitio web, tanto de texto, objetos e imágenes. Los archivos desarrollados en HTML usan la extensión .htm o .html. El lenguaje de HTML funciona por medio de "etiquetas" que describen la apariencia o función del texto enmarcado. Este lenguaje puede llegar a incluir un script o código que tenga incidencia en el comportamiento del navegador web de elección.

Caracteristicas de HTML:

  1. Las páginas HTML son documentos de texto.
  2. HTML usa elemntos para definir una estructura de la página.
  3. Los elementos se definen por etiquetas cerradas en parentesis angulares < y > .
  4. El contenido de un elemento se cierra entre etiquetas.
  5. Existen etiquetas de apertura < etiqueta > y cierre < /etiqueta > .
  6. El nombre de la etiqueta debe ser el mismo y se distingue a la etiqueta de cierre con una diagonal.

Elemento: < etiqueta > contenido < /etiqueta >

Las etiquetas pueden tener atributos, los cuales se especifican en la etiqueta de inicio.
Los atributos contienen información dicional acerca del contenido.

< etiqueta = atributo="valor" > contenido < /etiqueta >
